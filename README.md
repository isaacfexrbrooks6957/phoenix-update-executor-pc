## Overview

Phoenix Executor is a Windows-focused Roblox script executor built for Lua injection and fast local script execution. It aims to keep the desktop experience compact, so users can load scripts, browse the script hub, and handle Roblox script tasks without extra interface clutter.

Release 4.2 is centered on hands-on execution features: one-click injection, batch execution, offline script running, and a persistent queue for stored scripts. ## FAQ

### Is Phoenix Executor a Windows app?
Yes. It is built for Windows desktop use and is intended for Windows 10 and Windows 11.

### Does it support updates?
Yes. The project includes an auto-update engine intended to keep the app current with newer releases.

### How does it compare with other Roblox script executors?
Phoenix Executor leans into Lua injection, a sizable script hub, batch execution, and a local desktop interface instead of a browser-like layout.

### Is account risk something I should consider?
Any third-party script executor can carry platform and account risk. Use it carefully and follow the rules and terms that apply to your environment.

### Are scripts stored locally?
Yes. The app supports persistent queue behavior and offline execution, so local storage is part of the workflow.

### Will it always work with every Roblox update?
Compatibility may shift when Roblox or the surrounding environment changes. The auto-update engine is meant to help, but no compatibility guarantee should be assumed.
